## Step 1: Add TypeScript Interfaces

**File**: `src/lib/types.ts`

Append three new interfaces:

```ts
export interface ContributionDay {
  date: string;        // "2025-06-30"
  count: number;
  level: 0 | 1 | 2 | 3 | 4;
}

export interface ContributionWeek {
  days: ContributionDay[];
}

export interface ContributionData {
  totalContributions: number;
  weeks: ContributionWeek[];
  yearRange: string;   // "Jun 2024 — Jun 2025"
}
```

---

## Step 2: Add GitHub GraphQL Fetch

**File**: `src/lib/github.ts`

Append:

- `CONTRIBUTION_QUERY` — GraphQL query using `contributionsCollection.contributionCalendar` for the last year
- `fetchContributions()` — POSTs to `https://api.github.com/graphql` with Bearer token, returns raw calendar data
- `transformContributions(raw)` — Converts to `ContributionData`, computes levels (0-4) based on count thresholds:
  - 0 = empty, 1-3 = level 1, 4-6 = level 2, 7-9 = level 3, 10+ = level 4
- `getLevelColor(level)` — Returns CSS color using accent at varying opacities:
  - Level 0: `var(--color-surface)`, Level 1: `rgba(37,99,235,0.20)`, Level 2: `0.40`, Level 3: `0.65`, Level 4: `var(--color-accent)`
- Fallback function `generateFallbackData()` — 52 weeks of zero-count days

## Step 4: Create Contributions Section Component

**File**: `src/app/components/sections/Contributions.tsx` (NEW)

### Structure
```
<section id="contributions">
  <FadeIn>
    <SectionHeading eyebrow="Activity" title="Contributions" />
  </FadeIn>
  <FadeIn delay={0.1}>
    <p>Total N contributions in the last year</p>
  </FadeIn>
  <FadeIn delay={0.2}>
    <div overflow-x-auto>  ← horizontal scroll on mobile
      Month labels row
      Grid: day labels (Mon/Wed/Fri) + week columns of cells
      Legend: Less [squares] More
    </div>
  </FadeIn>
  <Tooltip />  ← fixed-position, follows mouse
</section>
```

### Heatmap Grid
- Cell size: 11px × 11px (matches GitHub standard)
- Gap: 3px between cells
- 7 rows (days), ~53 columns (weeks)
- Each cell is a `<button>` with `backgroundColor` from `getLevelColor(day.level)`
- Hover ring: `hover:ring-1 hover:ring-text-2/30`

### Tooltip
- `position: fixed`, follows mouse via `onMouseEnter`/`onMouseLeave`
- Shows: "N contributions" + formatted date
- Styled: `bg-surface border border-border text-text-1 shadow-lg`

### Data Fetching
- `useEffect` on mount calls `/api/contributions`
- Stores result in `useState<ContributionData | null>`
- On error, falls back to `generateFallbackData()`

### Responsive
- `overflow-x-auto no-scrollbar` on the grid container
- Mobile: horizontal scroll
- Desktop: fits within `max-w-[720px]`
